## 📖 Stage 1: Problem Statement & Motivation
With the exponential growth of digital communication, email spam remains a critical cybersecurity and productivity issue. Spam emails not only clutter inboxes but often carry malicious payloads such as phishing links or malware. 

The objective of this project is to build an automated, intelligent Email Spam Classification system. By leveraging **Natural Language Processing (NLP)** techniques to extract semantic meaning from raw email text, and applying **Machine Learning (ML)** algorithms, we aim to accurately categorize incoming emails as either "Spam" (malicious/junk) or "Ham" (legitimate).

**Dataset Chosen:** We are utilizing the **SpamAssassin Public Corpus**, which provides a robust, real-world collection of labeled spam and ham emails ideal for text classification tasks. 

**Methodology Overview:** Our pipeline includes:
1. **Text Preprocessing:** Tokenization, stop-word removal, and lemmatization.
2. **Feature Engineering:** TF-IDF and Word Embeddings.
3. **Model Training:** Comparing traditional ML (Naive Bayes, SVM) against Deep Learning (LSTM neural networks).
