# Model Training and Evaluation
---

## Objectives

- Train machine learning models
- Compare model performance
- Evaluate classification accuracy
- Select the best-performing model

---

## Feature Extraction

Text data was converted into numerical format using:

- TF-IDF Vectorizer

---

## Models Used

- Naive Bayes
- Logistic Regression
- Support Vector Machine (SVM)
- Random Forest

---

## Evaluation Metrics

The models were evaluated using:

- Accuracy
- Precision
- Recall
- F1-Score
- Confusion Matrix

---

## Results

| Model | Accuracy |
|---|---|
| Naive Bayes | High |
| Logistic Regression | High |
| SVM | Very High |
| Random Forest | High |

---

## Best Model

The best-performing model was selected based on:
- Highest accuracy
- Better precision and recall
- Low false positives

---

## Technologies Used

- Python
- Scikit-learn
- NumPy
- Pandas

---

## Conclusion

The trained models successfully classified spam and ham messages with high accuracy.
