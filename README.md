# Exploratory Data Analysis (EDA) - Email Spam Classifier

## Overview

This branch focuses on Exploratory Data Analysis (EDA) of the spam dataset to understand data distribution, patterns, and important insights before model development.

---

## Objectives

- Analyze dataset structure
- Identify spam and ham distribution
- Visualize text patterns
- Understand word frequency and message lengths

---

## Analysis Performed

### 1. Dataset Information
- Number of rows and columns
- Missing values
- Duplicate values

### 2. Class Distribution
Visualized the number of:
- Spam messages
- Ham messages

### 3. Message Length Analysis
Compared:
- Average spam message length
- Average ham message length

### 4. Word Frequency Analysis
Identified frequently used words in:
- Spam messages
- Ham messages

### 5. Visualization
Used:
- Bar Charts
- Histograms
- Word Clouds
- Count Plots

---

## Technologies Used

- Python
- Pandas
- Matplotlib
- Seaborn
- WordCloud

---

## Key Insights

- Spam messages usually contain promotional keywords
- Spam messages are often longer than ham messages
- Certain keywords appear frequently in spam texts
EDA helped understand the dataset and guided preprocessing and model-building decisions.
